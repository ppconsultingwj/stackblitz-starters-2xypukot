# P&P Consulting Website

Professional consulting website for process optimization, AI, and project management services.

## Features

- Modern, responsive design with Tailwind CSS
- Interactive navigation with mobile menu
- Training offerings with inquiry modals
- Blog section with expandable articles
- Contact form
- Smooth animations and transitions

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your images to the `public` folder:
   - `Właściciel P&P Consulting.jpg` - Owner photo
   - `Shogun Wojtek.webp` - Blog post 1 image
   - `shogun wojtek 2.webp` - Blog post 2 image
   - `zdjęcie biurka.webp` - Desk photo for blog post 2

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons
